Plant Monitor

Description

A device created to regularly measure and save the following parameters in order to monitor and control a plant's growth: air temperature, air relative humidity, soil moisture. The system is based on an Arduino Nano microcontroller. Time is measured by an RTC module. The air temperature sensor is a DS18B20, relative humidity of the air is measured by a DHT11 sensor. The data is saved on an SD card.
